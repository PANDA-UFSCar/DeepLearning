Basicamente temos aqui um projeto de Detector de Dígitos utilizando Python e React.

No programa python será necessário utilizar a biblioteca streamlit, podendo optar rodar pela imagem docker ou então subir a pasta cnn_python_streamlit no streamlithub deixando o app exposto na internet.
Com isso já seria o suficiente para fazer a CNN funcionar, entretanto, se quiser subir em um site utilizando IFrame, basta subir a pasta cnn_react em sua vps ou no vercel.

Dessa forma, o programa deve funcionar como mostrado na apresentação.
